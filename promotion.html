Additional Techniques for SEO

Map Your Site

An HTML site map is a web page with a map of the site that contains hierarchical list of hyperlinks to the major pages in your website. This can be helpful for website visitors and assist search engine robots as they follow hyperlinks on your site. An XML site map is an XML file that is used by search engines only, not visitors. An XML site map provides the following information about each page to a search engine:

  • date that each page was last modified
  • indicator of how frequently each page changes
  • priority level for each page

You can use an online site map generator to create a sitemap file.


Link Popularity

Search engines rate websites based on the number of sites that link to a particular website and the quality of those sites. Link popularity can determine its order in the SERP.


SMO - Social Media Optimization

Make it easy for your current and potential website visitors by creating content of value that is shareable. Such as providing links to Twitter, Facebook, and Pintrest.
